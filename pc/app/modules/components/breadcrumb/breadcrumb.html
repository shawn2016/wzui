breadcrumb 面包屑导航

在一个带有层次的导航结构中标明当前页面的位置,各路径间的分隔符已经自动通过 CSS 的 :before 里的 content 属性添加了

基本实例#

面包屑

  1. Home
  2. Library
  3. Data

面包屑icon

  1. Home
  2. Library
  3. Data
基本实例
名称 序号 效果 所需类名 定制所需 备注
面包屑 1 面包屑 .rob-breadcrumb(面包屑父标签) breadcrumb(必要) 配置$breadcrumb-separator参数来实现 > 或 / 隔断 ; 使用ol、ul定义一个.rob-breadcrumb类名
面包屑icon 2 面包屑icon .rob-breadcrumb(面包屑父标签) breadcrumb(必要) 配置$breadcrumb-separator参数来实现 > 或 / 隔断 ; 使用ol、ul定义一个.rob-breadcrumb类名 
        
    <!-- 面包屑默认 -->
    <ol class="rob-breadcrumb">
      <li><a href="#">Home</a></li>
      <li><a href="#">Library</a></li>
      <li class="active">Data</li>
    </ol>

    <!-- 面包屑 icon -->
    <ol class="rob-breadcrumb">
      <li><a href="#"><i class="rob-icon-home"></i>Home</a></li>
      <li><a href="#"><i class="rob-icon-home"></i>Library</a></li>
      <li  class="active"><a href="#">Data</a></li>
    </ol>
