Normalize 样式统一

在统一浏览器默认样式上，Reset 一度非常流行，更有简单粗暴的通配符 reset。

何时使用#

  • 时过境迁，Reset 逐渐淡出的前沿前端的视野，normalize.css 取而代之。

  • normalize.css，统一样式的同时保留可辨识性；

代码下载#

官方地址：http://necolas.github.io/normalize.css

其他途径

CDN:http://cdn.bootcss.com/normalize/6.0.0/normalize.css

压缩版:http://cdn.bootcss.com/normalize/6.0.0/normalize.min.css 

        
          <link rel="stylesheet" type="text/css" href="http://cdn.bootcss.com/normalize/6.0.0/normalize.min.css">

使用命令行进行安装，这样更为高逼格。

在线安装

首先安装node环境，使用npm，yarn都可以。 


              npm install normalize --save-dev
              或者
              yarn install normalize --save-dev
